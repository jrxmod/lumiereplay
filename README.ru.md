# Lumière Play — по-русски

Это дополнительный файл для тех кто предпочитает читать на русском.

---

## Что это

Мод добавляет блок-проектор. Ставишь его, кликаешь правой кнопкой, вставляешь ссылку на YouTube / Twitch / ВКонтакте или путь к локальному файлу — и на стене появляется виртуальный экран с видео. Работает в одиночной игре и на серверах, другие игроки видят и слышат то же самое.

---

## Как пользоваться

1. Скрафти Проектор Lumière
2. Поставь — он смотрит в ту сторону куда смотришь ты
3. Правый клик → вставь ссылку или путь к файлу
4. Выбери качество и размер экрана
5. Нажми Play

---

## Что нужно установить

На клиенте нужен VLC Media Player:

    Linux:   sudo pacman -S vlc   или   sudo apt install vlc
    Windows: https://www.videolan.org/vlc/
    macOS:   brew install vlc

yt-dlp скачается сам если его нет в системе.

На сервере ничего дополнительно не нужно.

---

## Поддерживаемые платформы

- YouTube (нужно быть залогиненным в браузере)
- Twitch — прямые трансляции и VOD
- Vimeo, Dailymotion, SoundCloud
- ВКонтакте / vkvideo.ru
- Rutube
- ok.ru/video
- Локальные файлы — абсолютный путь начиная с /

Яндекс.Видео не работает — сломан экстрактор в yt-dlp на их стороне.
